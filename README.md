**Building small AI tools · making electronic music as Total Reboot · shooting Tel Aviv sunsets and urban architecture**

`Python` · `Bash` · Claude Code skills · LLM / agent tooling · DSP & audio analysis (`librosa`, `Demucs`)

In music and photography, I'm after what isn't there yet — a sound that hasn't been heard, an image nature never assembles on its own. In the tools I build, the discipline flips: surface only what is measurably there. Whether it's the hidden dramaturgy of a techno track or the structural dead-ends in a product spec, I rely on exact measurements over estimated summaries. It's the same hunt either way: finding the connections that aren't obvious.
